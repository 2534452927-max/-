# Speech Recognition Test Dataset

## Description

This repository contains a small-scale Chinese speech dataset used for offline speech recognition evaluation.

## Dataset Content

- Chinese speech samples
- WAV format
- Used for model evaluation

## Evaluated Models

- Vosk
- Whisper
- VoxCPM

## Purpose

This dataset is used in the undergraduate graduation project:
"Design and Implementation of an Offline Speech Recognition System Based on Multi-Model Comparison".
